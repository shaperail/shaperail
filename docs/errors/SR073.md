# SR073

**Severity:** error

**Summary:** subscriber entry has an empty event pattern

## What this means

A `subscribers:` entry has an empty `event:` value. Subscriber event patterns must be non-empty strings — typically `resource.action` format or a wildcard like `*.deleted` — so the runtime knows which events to route to the handler.

## Example that triggers this

```yaml
endpoints:
  on_event:
    method: POST
    path: /items/on_event
    handler: on_event_handler
    subscribers:
      - event: ""
        handler: my_handler
```

## How to fix it

Provide a non-empty event pattern:

```yaml
subscribers:
  - event: items.created
    handler: my_handler
```
