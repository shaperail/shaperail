# AgentLedger — M01 Identity Backbone Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Scaffold the AgentLedger Shaperail project and ship the five identity-backbone resources (`organizations`, `users`, `customers`, `agents`, `tasks`) with auth roles, hierarchical agent/task trees, and a regenerable per-agent MCP secret — all validated via `shaperail check --json` and HTTP integration tests.

**Architecture:** A single Shaperail project named `agentledger`, scaffolded under `docs/example-project-shaperail.io/agentledger/`. Five YAML resource files in `resources/`, two controller files (`agents.controller.rs`, `tasks.controller.rs`), and one auth-config block in `shaperail.config.yaml`. Money fields are NOT introduced in M01. Postgres + Redis come from `docker compose` (auto-generated by `shaperail init`).

**Tech Stack:** Shaperail (Actix-web 4, sqlx, deadpool-redis, tokio), JWT auth via the `jsonwebtoken` crate (used by Shaperail), bcrypt or argon2 for password hashing, `hmac` + `sha2` for MCP-secret hashing, `assert_cmd` + `reqwest` for integration tests.

**Spec:** `docs/example-project-shaperail.io/docs/superpowers/specs/2026-05-01-agentledger-design.md` (sections 5, 11, 15.1).

**Branch:** `feat/agentledger-spec` (already created and contains the design spec).

## Conventions Used Throughout This Plan

- All commands assume CWD = `docs/example-project-shaperail.io/agentledger/` unless stated otherwise.
- Every task ends with a commit. Commit messages use `feat(m01): ...` or `chore(m01): ...` prefix.
- The implementer must run `shaperail check --json` before every commit to catch YAML errors early.
- Money fields are intentionally absent from M01. Do not introduce `amount_minor`, `currency`, or `cap_minor` here — those land in M02/M03.
- For nullable foreign keys (`parent_agent_id`, `parent_task_id`, `customer_id`), do NOT set `required: true`. Shaperail's `nullable:` flag is reserved for `timestamp`/`json` fields; non-required uuid fields default to nullable.
- Roles referenced in `auth:` lists must be declared in `shaperail.config.yaml`. The roles for M01 are `super_admin`, `admin`, `finance`, `viewer`. The `agent` role exists in config but no endpoint uses it in M01 (gateway endpoints arrive in M03). The `owner` role is built-in.

```rust
//! Shared test scaffolding: spawns the runtime against a test DB and mints JWTs.
use std::net::TcpListener;
use uuid::Uuid;

pub struct TestApp {
    pub address: String,
    pub client: reqwest::Client,
    jwt_secret: String,
}

impl TestApp {
    pub async fn spawn() -> Self {
        let listener = TcpListener::bind("127.0.0.1:0").expect("bind");
        let port = listener.local_addr().unwrap().port();
        let address = format!("http://127.0.0.1:{port}");
        let jwt_secret = std::env::var("JWT_SECRET").expect("JWT_SECRET must be set");

        // shaperail-runtime exposes a programmatic spawn helper.
        // If the API differs in the installed version, adapt the call below.
        let server = shaperail_runtime::test_support::spawn_with_listener(listener)
            .await
            .expect("spawn server");
        tokio::spawn(server);

        Self {
            address,
            client: reqwest::Client::new(),
            jwt_secret,
        }
    }

    pub fn url(&self, path: &str) -> String {
        format!("{}{}", self.address, path)
    }

    pub fn super_admin_jwt(&self) -> String {
        use jsonwebtoken::{encode, EncodingKey, Header};
        use serde_json::json;
        let claims = json!({
            "sub": Uuid::new_v4().to_string(),
            "role": "super_admin",
            "exp": (chrono::Utc::now().timestamp() + 3600) as usize,
        });
        encode(
            &Header::default(),
            &claims,
            &EncodingKey::from_secret(self.jwt_secret.as_bytes()),
        )
        .expect("encode jwt")
    }
}
```

```rust
//! Controller for the `users` resource.
use argon2::{password_hash::SaltString, Argon2, PasswordHasher};
use rand::rngs::OsRng;
use serde_json::Value;
use shaperail_runtime::ControllerContext;

/// Hash the plaintext `password` field and rewrite it as `password_hash`.
pub async fn hash_password(ctx: &mut ControllerContext) -> Result<(), String> {
    let password = ctx.input
        .get("password")
        .and_then(Value::as_str)
        .ok_or("password is required")?;

    if password.len() < 12 {
        return Err("password must be at least 12 characters".into());
    }

    let salt = SaltString::generate(&mut OsRng);
    let hash = Argon2::default()
        .hash_password(password.as_bytes(), &salt)
        .map_err(|e| format!("hash failed: {e}"))?
        .to_string();

    let obj = ctx.input
        .as_object_mut()
        .ok_or("input must be an object")?;
    obj.remove("password");
    obj.insert("password_hash".into(), Value::String(hash));
    Ok(())
}
```

```rust
//! Controllers for the `agents` resource.
//!
//! - `prepare_agent_create`: validates parent_agent_id is in the same org as
//!   the new agent (when set) and mints a fresh MCP secret, storing only the
//!   SHA-256 hash. The plaintext secret is returned in the response via the
//!   `ctx.set_response_extra` channel (one-time visibility).
//! - `regenerate_mcp_secret`: the same mint+hash flow without org validation.
use hmac::{Hmac, Mac};
use rand::rngs::OsRng;
use rand::RngCore;
use serde_json::Value;
use sha2::{Digest, Sha256};
use shaperail_runtime::ControllerContext;
use sqlx::Row;
use uuid::Uuid;

pub async fn prepare_agent_create(ctx: &mut ControllerContext) -> Result<(), String> {
    validate_parent_in_org(ctx).await?;
    mint_secret_into(ctx).await
}

pub async fn regenerate_mcp_secret(ctx: &mut ControllerContext) -> Result<(), String> {
    mint_secret_into(ctx).await
}

async fn validate_parent_in_org(ctx: &mut ControllerContext) -> Result<(), String> {
    let parent_id = match ctx.input.get("parent_agent_id").and_then(Value::as_str) {
        Some(s) if !s.is_empty() => s.to_string(),
        _ => return Ok(()), // no parent declared — root agent
    };
    let org_id = ctx.input
        .get("org_id")
        .and_then(Value::as_str)
        .ok_or("org_id required")?;
    let parent_uuid = Uuid::parse_str(&parent_id).map_err(|e| format!("parent_agent_id: {e}"))?;
    let org_uuid = Uuid::parse_str(org_id).map_err(|e| format!("org_id: {e}"))?;

    let row = sqlx::query("SELECT org_id FROM agents WHERE id = $1")
        .bind(parent_uuid)
        .fetch_optional(&ctx.db)
        .await
        .map_err(|e| format!("db error: {e}"))?
        .ok_or("parent_agent_id not found")?;

    let parent_org: Uuid = row.try_get("org_id").map_err(|e| format!("row: {e}"))?;
    if parent_org != org_uuid {
        return Err("parent_agent_id must belong to the same org".into());
    }
    Ok(())
}

async fn mint_secret_into(ctx: &mut ControllerContext) -> Result<(), String> {
    let mut bytes = [0u8; 32];
    OsRng.fill_bytes(&mut bytes);
    let secret = format!("aks_{}", base32::encode(base32::Alphabet::Rfc4648 { padding: false }, &bytes));

    let mut hasher = Sha256::new();
    hasher.update(secret.as_bytes());
    let hash_hex = format!("{:x}", hasher.finalize());

    let obj = ctx.input.as_object_mut().ok_or("input must be object")?;
    obj.insert("mcp_secret_hash".into(), Value::String(hash_hex));

    // Stash plaintext for the response builder.
    ctx.response_extras
        .insert("mcp_secret".into(), Value::String(secret));
    Ok(())
}
```

```rust
use serde_json::Value;
use shaperail_runtime::ControllerContext;
use sqlx::Row;
use uuid::Uuid;

pub async fn validate_task_invariants(ctx: &mut ControllerContext) -> Result<(), String> {
    let org_id = parse_uuid(&ctx.input, "org_id", true)?.unwrap();
    let agent_id = parse_uuid(&ctx.input, "agent_id", true)?.unwrap();
    let parent_id = parse_uuid(&ctx.input, "parent_task_id", false)?;
    let customer_id = parse_uuid(&ctx.input, "customer_id", false)?;

    if let Some(p) = parent_id {
        let row = sqlx::query("SELECT org_id FROM tasks WHERE id = $1")
            .bind(p)
            .fetch_optional(&ctx.db)
            .await
            .map_err(|e| format!("db: {e}"))?
            .ok_or("parent_task_id not found")?;
        let parent_org: Uuid = row.try_get("org_id").map_err(|e| format!("row: {e}"))?;
        if parent_org != org_id {
            return Err("parent_task_id must belong to the same org".into());
        }
    }

    let agent_row = sqlx::query("SELECT org_id FROM agents WHERE id = $1")
        .bind(agent_id)
        .fetch_optional(&ctx.db)
        .await
        .map_err(|e| format!("db: {e}"))?
        .ok_or("agent_id not found")?;
    let agent_org: Uuid = agent_row.try_get("org_id").map_err(|e| format!("row: {e}"))?;
    if agent_org != org_id {
        return Err("agent_id must belong to the same org".into());
    }

    if let Some(c) = customer_id {
        let cust_row = sqlx::query("SELECT org_id FROM customers WHERE id = $1")
            .bind(c)
            .fetch_optional(&ctx.db)
            .await
            .map_err(|e| format!("db: {e}"))?
            .ok_or("customer_id not found")?;
        let cust_org: Uuid = cust_row.try_get("org_id").map_err(|e| format!("row: {e}"))?;
        if cust_org != org_id {
            return Err("customer_id must belong to the same org".into());
        }
    }
    Ok(())
}

fn parse_uuid(v: &Value, key: &str, required: bool) -> Result<Option<Uuid>, String> {
    match v.get(key).and_then(Value::as_str) {
        Some(s) if !s.is_empty() => Uuid::parse_str(s)
            .map(Some)
            .map_err(|e| format!("{key}: {e}")),
        _ if required => Err(format!("{key} required")),
        _ => Ok(None),
    }
}
```
